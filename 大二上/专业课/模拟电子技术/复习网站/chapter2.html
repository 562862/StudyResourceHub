一、放大电路的基本概念

🔹 什么是放大？

放大是指在输入信号的控制下，将直流电源的能量转换为按输入信号变化规律变化的较大能量的过程。

注意：放大的不是能量（能量来自电源），而是信号！

🔹 放大的本质

关键要素：

  • 需要直流电源提供能量
  • 需要放大器件（三极管或场效应管）
  • 输出信号的能量 > 输入信号的能量
  • 放大的是电压、电流或功率

二、静态工作点

2.1 什么是静态工作点？

🔹 静态与动态

静态：没有输入信号时，电路中各点的直流电压和电流

动态：有输入信号时，电路中各点的电压和电流会在静态值基础上变化

静态工作点（Q点）：用IBQ、ICQ、UCEQ表示

2.2 静态工作点的计算

静态工作点计算公式：
IBQ = (VCC - UBEQ) / Rb
ICQ = β · IBQ
UCEQ = VCC - ICQ · Rc

📝 例题1：静态工作点计算

题目：某放大电路在负载开路时的输出电压为4V，接入3kΩ的负载后输出电压降为3V，则此电路的输出电阻为（）。

A. 0.5kΩ　　B. 1kΩ　　C. 2kΩ　　D. 3kΩ

解析：

1. 负载开路时：Uo = 4V（此时输出电阻上无电流）

2. 接负载后：Uo' = 3V，负载RL = 3kΩ

3. 负载电流：IL = Uo' / RL = 3V / 3kΩ = 1mA

4. 输出电阻上的压降：ΔU = 4V - 3V = 1V

5. 输出电阻：Ro = ΔU / IL = 1V / 1mA = 1kΩ

答案：B

三、三种基本放大电路

3.1 共射放大电路（CE）

🔹 共射放大电路的特点

电路特征：输入信号加在基极，输出信号从集电极取出，发射极公共接地

性能特点：

  • 电压放大倍数：较大（Au = -β·Rc/rbe）
  • 输入电阻：中等（几kΩ）
  • 输出电阻：较大（约等于Rc）
  • 相位关系：输出与输入反相（相位差180°）

⚠️ 共射放大电路的输出电阻

重要结论：与空载相比，接上负载后，单管放大器的动态范围一定变小。

原因：负载分流使得输出电压降低，输出范围变小。

3.2 共集放大电路（CC）- 射极跟随器

🔹 共集放大电路的特点

电路特征：输入信号加在基极，输出信号从发射极取出，集电极公共接地

性能特点：

  • 电压放大倍数：接近1（Au ≈ 1）
  • 输入电阻：很大
  • 输出电阻：很小
  • 相位关系：输出与输入同相

⚠️ 射极跟随器的应用

射极输出器无放大（）的能力：

答案：电压

• 射极跟随器电压放大倍数接近1，几乎不放大电压
• 但可以放大电流和功率
• 主要用于阻抗匹配和缓冲隔离

3.3 共基放大电路（CB）

🔹 共基放大电路的特点

电路特征：输入信号加在发射极，输出信号从集电极取出，基极公共接地

性能特点：

  • 电压放大倍数：较大（与共射相近）
  • 输入电阻：很小
  • 输出电阻：较大
  • 相位关系：输出与输入同相

3.4 三种组态对比

组态 电压放大倍数 输入电阻 输出电阻 相位关系 应用
共射(CE) 大 中 大 反相 通用放大
共集(CC) ≈1 大 小 同相 阻抗变换
共基(CB) 大 小 大 同相 高频放大

四、放大电路的分析方法

4.1 图解法

🔹 图解法的应用

适用范围：分析非线性失真、确定最大不失真输出幅度

两条线：

  • 直流负载线：用于确定静态工作点Q
  • 交流负载线：用于分析动态特性

4.2 微变等效电路法

🔹 微变等效电路法的步骤

步骤1：画出微变等效电路

  • 电容开路→短路（交流分析时）
  • 直流电源短路
  • 三极管用h参数等效电路代替

步骤2：计算性能指标

  • 电压放大倍数 Au
  • 输入电阻 Ri
  • 输出电阻 Ro
共射放大电路的关键公式：
Au = -β·(Rc∥RL) / rbe
Ri = Rb∥rbe
Ro ≈ Rc

五、失真问题

5.1 饱和失真

🔹 饱和失真的特征

原因：静态工作点Q设置过高，三极管进入饱和区

现象：输出波形顶部被削平

解决：减小Rb，降低IBQ，使Q点下移

5.2 截止失真

🔹 截止失真的特征

原因：静态工作点Q设置过低，三极管进入截止区

现象：输出波形底部被削平

解决：增大Rb，提高IBQ，使Q点上移

📝 例题2：失真分析

题目：在NPN型单级共射放大电路中，若输入电压为正弦波形，输出波形出现了底部被削平的现象，这种失真是（）。

A. 饱和失真　　B. 截止失真　　C. 线性失真　　D. 交越失真

解析：

1. 观察失真位置：底部被削平

2. 判断失真类型：

  • 顶部失真 → 饱和失真
  • 底部失真 → 截止失真 ✓

3. 失真原因：静态工作点太低，三极管在负半周进入截止区

答案：B（截止失真）

六、共模抑制比（CMRR）

🔹 差分放大电路

作用：抑制零点漂移，放大差模信号，抑制共模信号

共模抑制比：CMRR = Ad / Ac

• Ad：差模电压放大倍数
• Ac：共模电压放大倍数
• CMRR越大，电路性能越好

📝 例题3：差分放大电路

题目：图示电路参数理想对称，晶体管的β均为50，UBEQ≈0.7V。试计算Re滑动端在中点时T1管和T2管的发射极静态电流IEQ，以及动态参数Au、Ri、Ro。

已知：VCC = +12V，VEE = -6V，Rb = 10kΩ，Rc = 10kΩ，Rw = 100Ω

解析：

1. 静态分析：

发射极总电流：IE = (|VEE| - UBE) / Re

每管的静态电流：IEQ = IE / 2

2. 动态分析：

差模电压放大倍数：Au = -β·Rc / (2rbe)

输入电阻：Ri = 2rbe

输出电阻：Ro = 2Rc

七、本章小结

🎯 核心知识点总结

  • 静态工作点：决定放大电路的工作状态，影响失真
  • 共射电路：电压放大倍数大，输出反相
  • 共集电路：电压放大倍数≈1，用于阻抗匹配
  • 共基电路：输入电阻小，适合高频放大
  • 失真类型：饱和失真（顶部）和截止失真（底部）

⚠️ 常见易错点

  • 混淆三种组态的性能特点
  • 不能正确判断失真类型
  • 静态工作点计算时忘记UBE的值
  • 微变等效电路分析时电容处理错误
