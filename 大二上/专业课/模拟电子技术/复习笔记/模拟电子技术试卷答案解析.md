## 一、填空题（45分）

**每空3分，共15空**

### 答案速览

| 题号 | 答案 |
|------|------|
| 1 | **空穴** |
| 2 | **电压** 控制 **电流** |
| 3 | **恒流区** |
| 4 | **无限大**、**0**、**无限大** |
| 5 | **不能** |
| 6 | **窄** |
| 7 | **会** |
| 8 | **\|AF\|=1**、**±2nπ**、**\|AF\|>1** |
| 9 | **输入电阻**、**变宽** |

---

### 详细解析

#### 1. P型半导体的多数载流子

**答案：** 空穴

**解析：**
- P型半导体中掺入了**三价元素**（如硼B）
- 产生大量**空穴**作为多数载流子
- 电子为少数载流子
- 导电主要依靠空穴的运动

---

#### 2. 场效应管的控制特性

**答案：** 电压控制电流

**解析：**
- 场效应管（FET）是**电压控制型**器件
- 栅源电压 $U_{GS}$ 控制漏极电流 $I_D$
- 输入阻抗极高（栅极几乎不取电流）
- 区别于BJT的电流控制型特性

---

#### 3. 结型场效应管的工作区域

**答案：** 恒流区（饱和区）

**解析：**
- 放大电路要求FET工作在**恒流区**
- 在恒流区，$I_D$ 基本不随 $V_{DS}$ 变化
- 具有良好的恒流特性和放大能力
- 可变电阻区用于开关和调制电路

---

#### 4. 理想运算放大器的参数

**答案：** 输入电阻=**无限大**、输出电阻=**0**、放大倍数=**无限大**

**解析：**
理想运放的三大特性：

| 参数 | 理想值 | 实际意义 |
|------|--------|----------|
| 输入电阻 $R_i$ | $\infty$ | "虚断"：输入端不取电流 |
| 输出电阻 $R_o$ | 0 | 带负载能力强，输出不受负载影响 |
| 开环增益 $A_{od}$ | $\infty$ | "虚短"：差模输入电压趋于0 |

---

#### 5. 阻容耦合电路对缓变信号的放大能力

**答案：** 不能

**解析：**
- 阻容耦合电路中的**耦合电容**和**旁路电容**对直流和极低频呈高阻抗
- 频率极低的缓变信号无法通过耦合电容
- 会被耦合电容阻断或严重衰减
- 只能放大中高频交流信号

```
信号频率特性：
  通过能力
    │
    │     ┌─────────────── 通频带
    │    ╱│               │╲
  1 │───╱─┼───────────────┼─╲───
    │  ╱  │               │  ╲
    │ ╱   │               │   ╲
  0 └─────┴───────────────┴────┴──> 频率
         fL              fH
        (低频)          (高频)
         ↑
    缓变信号被阻断
```

---

#### 6. 多级放大电路的通频带

**答案：** 窄

**解析：**
多级放大时，频带收窄的原因：

$$f_{H总} = \min(f_{H1}, f_{H2}, ..., f_{Hn})$$

$$f_{L总} = \max(f_{L1}, f_{L2}, ..., f_{Ln})$$

$$BW_{总} = f_{H总} - f_{L总} < BW_{单级}$$

- 上限频率取最小值（高频特性恶化）
- 下限频率取最大值（低频特性恶化）
- 通频带 $BW$ 变窄

---

#### 7. 信号源频率变化对电路性能的影响

**答案：** 会

**解析：**
放大电路的性能参数均与频率相关：

| 参数 | 频率影响 |
|------|----------|
| 电压增益 $A_u$ | 频率↑→增益↓ |
| 相位 $\varphi$ | 频率↑→相移↑ |
| 输入阻抗 $R_i$ | 频率变化时改变 |
| 输出阻抗 $R_o$ | 频率变化时改变 |

工作频率改变会导致：
- 增益变化
- 相位失真
- 阻抗特性改变

---

#### 8. 正弦波振荡的条件

**答案：** 幅度平衡条件 $|AF|=1$，相位平衡条件 $\arg AF = \pm 2n\pi$，起振条件 $|AF|>1$

**解析：**

**平衡条件（维持振荡）：**

$$\begin{cases}
|AF| = 1 & \text{（幅度平衡）} \\
\arg AF = \pm 2n\pi & \text{（相位平衡，}n=0,1,2,...\text{）}
\end{cases}$$

**起振条件（开始振荡）：**

$$|AF| > 1$$

```
振荡过程：
  |AF|
    │
  1 │─────●───────────  稳定振荡
    │    ╱ ↓限幅
    │   ╱
    │  ╱  起振
    │ ╱   |AF|>1
    │╱
  0 └──────────────> t
```

- 起振阶段：$|AF|>1$，振幅逐渐增大
- 限幅阶段：非线性元件使 $|AF|$ 减小
- 稳定阶段：$|AF|=1$，等幅振荡

---

#### 9. 串联负反馈的影响

**答案：** 增大**输入电阻**，通频带**变宽**

**解析：**

**输入电阻增大：**

$$R_{if} = R_i(1 + AF)$$

- 串联负反馈使输入阻抗增大
- 减小对信号源的负载效应

**通频带展宽：**

$$BW_f = BW \cdot (1 + AF)$$

$$f_{Hf} = f_H \cdot (1 + AF)$$

$$f_{Lf} = \frac{f_L}{1 + AF}$$

```
频带展宽示意：
  |A|
    │
    │  无反馈
    │    ╱─╲
    │   ╱   ╲
    │  ╱     ╲
    │ ╱  有负反馈 ╲
    │╱─────────────╲
    └─┬─────────┬───┬──> f
     fLf      fL fH fHf
      └─────┬─────┘
         展宽后
```

## 二、电路分析题（10分）

### 题目

**电路如图所示，设输入 $u_i(t) = 12\cos 314t$ (V)**

已知：
- 二极管为理想二极管
- 电路为桥式整流+电容滤波
- 滤波电容 $C = 6\mu F$
- 负载电阻 $R_L = 200\Omega$
- 输入频率 $f = \omega/(2\pi) = 314/(2\pi) = 50Hz$

要求：
1. 绘出 $u_o(t)$ 的波形（5分）
2. 计算输出电压的平均值（5分）

---

### 解答

#### （1）输出波形 $u_o(t)$（5分）

**电路分析：**

这是一个**桥式整流+电容滤波电路**：

```
输入 ui(t) ──┬──D1──┬──(+)─┬── uo(t)
            │      │      │
            D4     C      RL
            │    6μF    200Ω
            │      │      │
            └──D3──┴──(-)─┴── GND
                  桥式整流
```

**工作原理：**

1. **整流阶段**：桥式整流将交流转换为全波脉动直流
2. **滤波阶段**：电容充放电平滑输出电压

**波形参数计算：**

输入电压有效值：
$$U_i = 12V$$

输出峰值电压：
$$U_{om} = \sqrt{2} \times U_i = 1.414 \times 12 = 16.97V \approx 17V$$

纹波电压峰峰值：
$$\Delta U = \frac{U_{om}}{f \cdot C \cdot R_L} = \frac{17}{50 \times 6 \times 10^{-6} \times 200} = \frac{17}{0.06} = 2.83V$$

输出平均值：
$$U_o = U_{om} - \frac{\Delta U}{2} = 17 - 1.42 = 15.58V$$

**波形图：**

```
uo(V)
  17 ├─╲    ╱╲    ╱╲    ╱╲    ╱╲
     │  ╲  ╱  ╲  ╱  ╲  ╱  ╲  ╱  ╲
     │   ╲╱    ╲╱    ╲╱    ╲╱    ╲╱   锯齿波
15.6 ├────●─────●─────●─────●────────  平均值
     │
  14 ├──────────────────────────────
     │
     │  ↑充电快 ↓放电慢
     │
   0 └─────┬─────┬─────┬─────┬────> t(ms)
           10    20    30    40
           │     │     │     │
        (放电)(充电)(放电)(充电)
```

**波形特征（得分要点）：**

| 特征 | 说明 | 分值 |
|------|------|------|
| 波形类型 | 脉动直流，锯齿波形状 | 1分 |
| 峰值电压 | $U_{om} = 17V$ | 1分 |
| 输出频率 | 100Hz（桥式整流后频率翻倍） | 1分 |
| 充放电特性 | 快速上升（充电）+ 缓慢下降（放电） | 1分 |
| 纹波大小 | $\Delta U \approx 2.83V$ | 1分 |

---

#### （2）输出电压平均值（5分）

**方法一：纹波法（推荐）**

桥式整流电容滤波电路的输出平均值：

$$U_o = U_{om} - \frac{\Delta U}{2}$$

**步骤1：** 计算输出峰值
$$U_{om} = \sqrt{2} U_i = \sqrt{2} \times 12 = 16.97V$$

**步骤2：** 计算纹波电压
$$\Delta U = \frac{U_{om}}{f C R_L}$$

其中：
- $f = 50Hz$（输入频率）
- $C = 6\mu F = 6 \times 10^{-6}F$
- $R_L = 200\Omega$

$$\Delta U = \frac{16.97}{50 \times 6 \times 10^{-6} \times 200} = \frac{16.97}{0.06} = 2.83V$$

**步骤3：** 计算平均值
$$U_o = 16.97 - \frac{2.83}{2} = 16.97 - 1.42 = 15.55V$$

---

**方法二：经验公式法**

对于桥式整流电容滤波电路：

$$U_o = U_{om}\left(1 - \frac{1}{2fCR_L}\right)$$

代入数据：

$$U_o = 16.97 \times \left(1 - \frac{1}{2 \times 50 \times 6 \times 10^{-6} \times 200}\right)$$

$$= 16.97 \times \left(1 - \frac{1}{0.12}\right)$$

$$= 16.97 \times (1 - 8.33)$$

等等，这个公式有误，应该是：

$$U_o = U_{om}\left(1 - \frac{1}{2fCR_L}\right) = 16.97 \times \left(1 - \frac{1}{2 \times 50 \times 6 \times 10^{-6} \times 200}\right)$$

让我重新计算：

$$\frac{1}{2fCR_L} = \frac{1}{2 \times 50 \times 6 \times 10^{-6} \times 200} = \frac{1}{0.12} = 8.33$$

这不对，应该是放电时间常数的概念。

**正确的经验公式：**

$$U_o \approx U_{om} - \frac{I_L}{fC} = U_{om} - \frac{U_{om}}{fCR_L}$$

$$= U_{om}\left(1 - \frac{1}{fCR_L}\right)$$

不对，标准公式应该是：

$$U_o = (1.2 \sim 1.4) U_i$$

对于 $RL \cdot C \gg T/2$ 的情况：

$$U_o \approx 1.2 U_i = 1.2 \times 12 = 14.4V$$

但这是估算值，精确值应该用第一种方法。

---

**计分点分配：**

| 步骤 | 内容 | 分值 |
|------|------|------|
| 1 | 计算峰值电压 $U_{om} = 16.97V$ | 1分 |
| 2 | 识别参数 $f=50Hz, C=6\mu F, R_L=200\Omega$ | 1分 |
| 3 | 计算纹波 $\Delta U = 2.83V$ | 2分 |
| 4 | 计算平均值 $U_o = 15.6V$ | 1分 |

---

**最终答案：**

$$\boxed{U_o = 15.6V}$$

## 三、共射放大电路分析（10分）

### 题目

**电路参数：**
- $U_{CC} = 12V$
- $R_C = 3k\Omega$
- $R_1 = 6k\Omega$
- $R_2 = 600\Omega$
- $U_{BE} = 0.6V$
- $R_w = 200\Omega$（发射极电阻）
- 晶体管特性曲线见图2

**要求：**
1. 计算静态基极电流和集电极电流（4分）
2. 计算电压放大倍数（2分）
3. 设计一个两级阻容耦合共射放大电路（4分）

---

### 解答

#### （1）静态工作点计算（4分）

**电路分析：**

这是一个**分压式偏置电路**：

```
        VCC=12V
         │
    ┌────┼────┬────┐
    │    │    │    │
   R1   R2   RC   │
   6k   600  3k   │
    │    │    │   │
    ├────●────┤   ├──○ uo
    │    │    │   │
    │   ┌┴┐  ┌┴┐  │
    │   │ │  │C│  │
    │   │B├──┤ │  │
    │   │ │  │E│  │
    │   └┬┘  └┬┘  │
    │    │    │   │
    │    │   RE   │
    │    │   200  │
   GND   │    │  GND
        GND  GND
```

**步骤1：计算基极电压 $U_B$**（1分）

分压公式：

$$U_B = \frac{R_2}{R_1 + R_2} \times U_{CC}$$

$$= \frac{600}{6000 + 600} \times 12$$

$$= \frac{600}{6600} \times 12 = 0.0909 \times 12 = 1.09V$$

---

**步骤2：计算发射极电压 $U_E$**

$$U_E = U_B - U_{BE} = 1.09 - 0.6 = 0.49V$$

---

**步骤3：计算发射极电流 $I_E$**（1分）

$$I_E = \frac{U_E}{R_E} = \frac{0.49}{200} = 2.45mA$$

---

**步骤4：计算集电极电流 $I_C$**（1分）

由于 $\beta$ 较大（从图2估计 $\beta \approx 60$）：

$$I_C \approx I_E = 2.45mA \approx 2.4mA$$

---

**步骤5：计算基极电流 $I_B$**（1分）

从图2的输出特性曲线查找：
- 当 $I_C = 2.4mA$ 时
- 对应的基极电流约为 $I_B = 40\mu A$

或者用公式计算（假设 $\beta = 60$）：

$$I_B = \frac{I_C}{\beta} = \frac{2.4mA}{60} = 40\mu A$$

---

**验证静态工作点：**

集电极电压：
$$U_C = U_{CC} - I_C R_C = 12 - 2.4 \times 10^{-3} \times 3000 = 12 - 7.2 = 4.8V$$

集射极电压：
$$U_{CE} = U_C - U_E = 4.8 - 0.49 = 4.31V$$

工作在放大区（$U_{CE} > U_{BE}$），满足条件✓

---

**最终答案：**

$$\boxed{I_B = 40\mu A, \quad I_C = 2.4mA}$$

---

#### （2）电压放大倍数（2分）

**微变等效电路参数：**

**输入电阻 $r_{be}$：**

$$r_{be} = r_{bb'} + (1 + \beta) \frac{26mV}{I_E}$$

$$= 300 + (1 + 60) \times \frac{26}{2.4}$$

$$= 300 + 61 \times 10.83 = 300 + 660.6 = 960.6\Omega$$

---

**电压放大倍数 $A_u$：**

假设发射极电阻 $R_E$ 被旁路电容短路（交流短路）：

$$A_u = -\frac{\beta R_C}{r_{be}}$$

$$= -\frac{60 \times 3000}{960.6}$$

$$= -\frac{180000}{960.6} = -187.4$$

---

**考虑负载后的增益：**

如果输出端接负载 $R_L$：

$$A_u = -\frac{\beta (R_C \parallel R_L)}{r_{be}}$$

假设空载：

$$\boxed{A_u \approx -187}$$

**负号表示输出与输入反相**

---

## 四、频率特性分析（10分）

### 题目

已知放大电路的电压放大倍数为：

$$A = \frac{30f^2}{(1 + jf)\left(1 + j\frac{f}{5 \times 10^3}\right)^2}$$

要求：
1. 求中频放大倍数大小和相位（6分）
2. 画出对数幅频和对数相频特性（伯德图）（4分）

---

### 解答

#### （1）传递函数分析

**标准化形式：**

将传递函数改写为：

$$A(jf) = \frac{30f^2}{(1 + jf)(1 + j\frac{f}{5000})^2}$$

**零极点分布：**

- **零点**：2个零点在原点 $f = 0$（二阶高通特性）
- **极点**：

- $(1 + jf) \approx jf$ （因为 $f \gg 1Hz$）
- $(1 + j\frac{f}{5000}) \approx 1$ （因为 $f \ll 5kHz$）

$$\varphi_{mid} = 180^\circ - 90^\circ - 0^\circ = 90^\circ$$

#### （1）最大输出功率及效率（4分）

其中：
- $V_{CC} = 16V$：电源电压
- $U_{CES} = 2V$：晶体管饱和压降

---

**步骤2：** 计算最大输出功率（1分）

$$P_{om} = \frac{U_{om}^2}{2R_L}$$

$$= \frac{14^2}{2 \times 4}$$

$$= \frac{196}{8} = 24.5W$$

---

**最大效率 $\eta_{max}$：**

**理论最大效率：**（1分）

OTL电路的理论最大效率：

$$\eta_{max} = \frac{\pi}{4} \times 100\% = 0.785 \times 100\% = 78.5\%$$

---

**实际效率（考虑饱和压降）：**（1分）

**步骤1：** 计算最大输出电流幅值

$$I_{om} = \frac{U_{om}}{R_L} = \frac{14}{4} = 3.5A$$

**步骤2：** 计算电源提供的平均功率

$$P_V = \frac{2V_{CC} \cdot I_{om}}{\pi}$$

$$= \frac{2 \times 16 \times 3.5}{\pi}$$

$$= \frac{112}{3.14159} = 35.65W$$

**步骤3：** 计算实际效率

$$\eta = \frac{P_{om}}{P_V} \times 100\%$$

$$= \frac{24.5}{35.65} \times 100\%$$

$$= 68.7\%$$

---

**最终答案：**

$$\boxed{P_{om} = 24.5W}$$

$$\boxed{\eta_{max} = 78.5\% \text{（理论）}, \quad \eta = 68.7\% \text{（实际）}}$$

#### （2）晶体管的最大功耗（2分）

对于OTL电路，单管最大功耗发生在输出功率约为最大功率的40%时。

**计算公式：**（1分）

$$P_{Tmax} = \frac{(V_{CC} - U_{CES})^2}{2\pi^2 R_L}$$

**代入数据：**（1分）

$$P_{Tmax} = \frac{14^2}{2 \times \pi^2 \times 4}$$

$$= \frac{196}{2 \times 9.8696 \times 4}$$

$$= 2.48W \approx 2.5W$$

**工程选择建议：**

实际选择晶体管时，应考虑1.5~2倍的安全裕量：

**建议选择：** $P_T \geq 5W$ 的功率管

#### （3）两个二极管的作用（4分）

**作用一：消除交越失真**（2分）

**问题分析：**

- 使Q1、Q2始终处于**微导通**状态
- 消除死区，避免交越失真

**作用二：温度补偿**（2分）

**温度特性问题：**

晶体管的 $U_{BE}$ 具有**负温度系数**：

$$\frac{dU_{BE}}{dT} \approx -2\,\frac{\mathrm{mV}}{{}^\circ\mathrm{C}}$$

**温度升高时的影响：**

| 参数 | 变化 | 后果 |
|------|------|------|
| $U_{BE}$ | ↓减小 | $I_C$增大 |
| $I_C$ | ↑增大 | 功耗增大，温度进一步升高 |
| 温度 | ↑升高 | 形成正反馈 |
| 最终 | 失控 | **热击穿** |

---

**温度补偿原理：**

二极管与晶体管具有**相似的温度特性**：

$$\frac{dU_D}{dT} \approx -2\,\frac{\mathrm{mV}}{{}^\circ\mathrm{C}}$$

**补偿过程：**

1. 温度升高 → 晶体管 $U_{BE}$ ↓ → $I_C$ 趋向增大
2. 温度升高 → 二极管 $U_D$ ↓ → 偏置电压↓ → $I_C$ 趋向减小
3. 两者相互抵消，**稳定静态工作点**

**数学表达：**

温度变化 $\Delta T$ 时：

$$\Delta U_{BE} = -2\,\frac{\mathrm{mV}}{{}^\circ\mathrm{C}} \times \Delta T$$

$$\Delta U_D = -2\,\frac{\mathrm{mV}}{{}^\circ\mathrm{C}} \times \Delta T$$

$$\Delta(U_D - U_{BE}) \approx 0$$

---

**工程实现要点：**

**关键要求：** 二极管与功放管必须**热耦合**

- 安装在**同一散热器**上
- 保证两者温度同步变化
- 才能实现有效的温度补偿

```
散热器布局示意：

    ┌──────────────────┐
    │   散热器         │
    │                  │
    │  ┌──┐  ┌──┐     │
    │  │Q1│  │D1│     │← 热耦合
    │  └──┘  │D2│     │
    │        └──┘     │
    │  ┌──┐          │
    │  │Q2│          │
    │  └──┘          │
    └──────────────────┘
```

**总结（4分）：**

| 作用 | 说明 | 分值 |
|------|------|------|
| **消除交越失真** | 提供1.4V偏置，使Q1、Q2微导通 | 2分 |

**关键点：**
- 偏置电压 = $U_{D1} + U_{D2} \approx U_{BE1} + |U_{BE2}|$
- 温度系数匹配：$\frac{dU_D}{dT} \approx \frac{dU_{BE}}{dT}$

### 解答

#### （1）减法器电路设计（7分）

**标准减法器电路原理：**

减法器（差分放大器）要求实现：

$$u_o = K(u_1 - u_2)$$

其中 $K$ 为放大系数，本题要求 $K = 1$。

**设计条件：**

标准减法器需满足：

1. 反相输入支路：$\frac{R_2}{R_1} = K$
2. 同相输入支路：电阻匹配条件 $\frac{R_1}{R_2} = \frac{R_3}{R_4}$

---

**电路设计：**

```

      u₁ ───────────/\/\/\───────┐

                      │

• u₁ 通过 R₂ 接运放反相输入端(-)
• 反相输入端通过 R₁ 接地
```

| $R_4$ | 10kΩ | 同相输入下分压电阻（需外加） |

---

**输出表达式推导：**（2分）

**同相端电压：**

$$u_+ = \frac{R_4}{R_3 + R_4} \cdot u_2 = \frac{10k}{10k + 10k} \cdot u_2 = 0.5u_2$$

**根据"虚短"：**

$$u_- = u_+ = 0.5u_2$$

**反相端节点电流方程（KCL）：**

流入反相端的电流：

$$i_1 = \frac{u_1 - u_-}{R_2} = \frac{u_1 - 0.5u_2}{5k}$$

流出反相端的电流：

$$i_2 = \frac{u_- - u_o}{R_1} = \frac{0.5u_2 - u_o}{5k}$$

根据"虚断"（$i_- = 0$）：

$$i_1 = i_2$$

$$\frac{u_1 - 0.5u_2}{5k} = \frac{0.5u_2 - u_o}{5k}$$

$$u_1 - 0.5u_2 = 0.5u_2 - u_o$$

$$u_o = u_2 - u_1$$

**判断方法：** 瞬时极性法

**分析：**

假设输出 $u_o$ 瞬时增大：

$$u_o ↑ \rightarrow u_- ↑ \text{（通过 } R_2 \text{ 反馈）}$$

$$u_- ↑ \rightarrow u_o ↓ \text{（反相放大）}$$

输出增大导致输出减小，形成负反馈。

**结论：** ✅ **负反馈**

---

**维度3：电压反馈 / 电流反馈**（1分）

**判断方法：** 看反馈信号取自何处

| 类型 | 取样点 | 特点 |
|------|--------|------|
| **电压反馈** | 输出电压 $u_o$ | 稳定输出电压，输出阻抗↓ |
| **电流反馈** | 输出电流 $i_o$ | 稳定输出电流，输出阻抗↑ |

**分析：**
- 反馈信号取自输出端电压 $u_o$
- 与负载电流无关
- 反馈信号正比于 $u_o$

**结论：** ✅ **电压反馈**

---

**维度4：串联反馈 / 并联反馈**（2分）

**判断方法：** 看反馈信号与输入信号的叠加方式

| 类型 | 连接方式 | 输入阻抗 |
|------|----------|----------|
| **串联反馈** | 电压求和 | $R_i ↑$ |
| **并联反馈** | 电流求和 | $R_i ↓$ |

**分析方法1（看输入端）：**

- 反相输入端同时连接：
  - 输入电阻 $R_1$（接地）
  - 反馈电阻 $R_2$（接输出）
- 反馈电流与输入电流在反相端节点汇合
- 属于**电流求和**方式

**分析方法2（看连接拓扑）：**

```
      u₂ ──── R₂ ────●──── u_-
                     │
      GND ─── R₁ ────┘

电流汇合点，并联反馈
```

**结论：** ✅ **并联反馈**

---

**综合判断总结：**

| 维度 | 判断结果 | 分值 |
|------|----------|------|
| 交/直流 | **交直流反馈** | 1分 |
| 正/负 | **负反馈** | 1分 |
| 电压/电流 | **电压反馈** | 1分 |
| 串联/并联 | **并联反馈** | 2分 |

---

**最终答案：**

$$\boxed{\text{交直流电压并联负反馈}}$$

**简写：** DC-AC Voltage Shunt Negative Feedback

---

**反馈效果：**

1. **稳定输出电压**（电压反馈）
2. **降低输入阻抗**（并联反馈）
3. **降低输出阻抗**（电压反馈）
4. **展宽通频带**（负反馈）
5. **提高线性度**（负反馈）

## 📚 补充内容：常见错误与解题技巧

### 一、常见错误分析

#### 填空题常见错误

**1. P型/N型半导体混淆**
- ❌ 错误：认为P型半导体中电子是多数载流子
- ✅ 正确：P型→空穴为多数载流子，N型→电子为多数载流子
- 💡 记忆技巧：**P→Positive→空穴带正电**

**2. 场效应管工作区域混淆**
- ❌ 错误：认为放大电路工作在可变电阻区
- ✅ 正确：放大电路工作在**恒流区（饱和区）**
- 💡 区分要点：
  - 可变电阻区：$V_{DS}$ 小，$I_D$ 随 $V_{DS}$ 变化（开关应用）
  - 恒流区：$V_{DS}$ 大，$I_D$ 基本不变（放大应用）

**3. 理想运放参数记错**
- ❌ 错误：输入电阻=0，输出电阻=∞
- ✅ 正确：输入电阻=∞，输出电阻=0
- 💡 记忆口诀：**"输入无穷大（虚断），输出零阻抗"**

**4. 振荡条件混淆**
- ❌ 错误：平衡条件 |AF|>1
- ✅ 正确：**起振** |AF|>1，**平衡** |AF|=1
- 💡 理解：起振时需要增益储备，稳定后自动调整到平衡

---

#### 整流滤波电路常见错误

**1. 频率计算错误**
- ❌ 错误：桥式整流后输出频率仍为50Hz
- ✅ 正确：桥式整流后频率**翻倍**为100Hz
- 💡 原因：全波整流，正负半周都整流

**2. 纹波公式记错**
- ❌ 错误：$\Delta U = \frac{I_L}{C}$
- ✅ 正确：$\Delta U = \frac{U_{om}}{fCR_L}$ 或 $\frac{I_L}{fC}$
- 💡 关键：别忘了频率 $f$

**3. 峰值计算错误**
- ❌ 错误：$U_{om} = U_i$（用有效值）
- ✅ 正确：$U_{om} = \sqrt{2}U_i$（要转换成峰值）
- 💡 记住：电容充电到峰值电压

---

#### 放大电路常见错误

**1. β值使用不当**
- ❌ 错误：直接用 $\beta = 100$ 计算，不看图
- ✅ 正确：从**输出特性曲线**读取或用给定值
- 💡 提醒：优先使用题目给定的曲线或参数

**2. rbe计算漏项**
- ❌ 错误：$r_{be} = (1+\beta)\frac{26mV}{I_E}$
- ✅ 正确：$r_{be} = r_{bb'} + (1+\beta)\frac{26mV}{I_E}$
- 💡 别忘：基区体电阻 $r_{bb'} \approx 300\Omega$

**3. 增益符号错误**
- ❌ 错误：共射放大器 $A_u = +\frac{\beta R_C}{r_{be}}$
- ✅ 正确：共射放大器 $A_u = -\frac{\beta R_C}{r_{be}}$（反相）
- 💡 记忆：**共射反相，共集同相，共基同相**

**4. 静态工作点验证遗漏**
- ❌ 错误：算完就结束，不验证
- ✅ 正确：必须验证 $U_{CE} > U_{BE}$（放大区条件）
- 💡 养成习惯：每次都验证工作区

---

#### 频率特性常见错误

**1. 中频段判断错误**
- ❌ 错误：任意选一个频率点
- ✅ 正确：中频段满足 $f_L \ll f \ll f_H$
- 💡 典型选择：**几何中心** $f_m = \sqrt{f_L \cdot f_H}$

**2. 相位计算符号错误**
- ❌ 错误：零点 -90°，极点 +90°
- ✅ 正确：**零点 +90°，极点 -90°**
- 💡 记忆：零点"提升"（+），极点"拉低"（-）

**3. 伯德图斜率计算错误**
- ❌ 错误：二阶极点斜率 -20dB/dec
- ✅ 正确：二阶极点斜率 **-40dB/dec**
- 💡 规律：n阶零点 +20n dB/dec，n阶极点 -20n dB/dec

---

#### 功率放大器常见错误

**1. 效率公式混淆**

- ❌ 错误：甲类、乙类、OTL效率不分
- ✅ 正确：
  - 甲类：$\eta_{max} = 50\%$
  - 乙类（OCL）：$\eta_{max} = 78.5\%$
  - OTL：$\eta_{max} = 78.5\%$
- 💡 记忆：乙类和OTL效率相同 $\frac{\pi}{4}$

**2. 管耗计算时机错误**
- ❌ 错误：认为 $P_T$ 在 $P_o = P_{om}$ 时最大
- ✅ 正确：$P_T$ 在 $P_o \approx 0.4P_{om}$ 时最大
- 💡 原因：此时管子损耗最大（既有电压又有电流）

**3. 二极管作用不全面**
- ❌ 错误：只答"消除交越失真"
- ✅ 正确：**1)消除交越失真 2)温度补偿**（两者都要答）
- 💡 提醒：每个2分，缺一个扣2分

---

#### 运放电路常见错误

**1. 输入对调导致符号错误**
- ❌ 错误：设计 $u_o = u_1 - u_2$ 时输入接错
- ✅ 正确：
  - $u_1$ 接**同相端**分压网络
  - $u_2$ 接**反相端**输入
- 💡 验证：推导后检查输出表达式

**2. 反馈类型判断错误**
- ❌ 错误：串并联反馈判断混乱
- ✅ 正确：看输入端连接方式
  - **串联**：反馈信号与输入信号**电压叠加**
  - **并联**：反馈信号与输入信号**电流汇合**
- 💡 技巧：反相输入端→并联，同相输入端→串联

**3. 工作状态判断依据不足**
- ❌ 错误：只说"线性区"，不说理由
- ✅ 正确：说明**存在负反馈** + **虚短虚断成立**
- 💡 完整答案：深度负反馈→线性区→虚短虚断

### 三、知识点扩展

#### 扩展1：半导体物理深化

**载流子浓度关系：**

对于本征半导体：
$$n_i = p_i$$

对于P型半导体（掺杂浓度$N_A$）：
$$p_p \approx N_A \quad (\text{多数载流子})$$
$$n_p = \frac{n_i^2}{p_p} \quad (\text{少数载流子})$$

对于N型半导体（掺杂浓度$N_D$）：
$$n_n \approx N_D \quad (\text{多数载流子})$$
$$p_n = \frac{n_i^2}{n_n} \quad (\text{少数载流子})$$

**质量作用定律：**
$$n \cdot p = n_i^2 = \text{常数}$$

#### 扩展2：BJT与FET对比

| 特性 | BJT（双极型） | FET（场效应） |
|------|---------------|---------------|
| 控制方式 | 电流控制 $I_B$ | 电压控制 $U_{GS}$ |
| 输入阻抗 | 低（kΩ级） | 极高（MΩ级） |
| 载流子 | 双极（电子+空穴） | 单极（电子或空穴） |
| 温度稳定性 | 较差（$\beta$随温度变化大） | 好 |
| 制造工艺 | 复杂 | 简单（适合集成） |
| 噪声 | 较大 | 小 |
| 应用场合 | 功率放大 | 高输入阻抗、开关电路 |

#### 扩展5：运放应用电路大全

**1. 基本运算电路**

```
反相放大器：Uo = -(Rf/R1)·Ui
同相放大器：Uo = (1+Rf/R1)·Ui
电压跟随器：Uo = Ui
加法器：    Uo = -(Rf/R1·U1 + Rf/R2·U2)
减法器：    Uo = (Rf/R1)·(U1 - U2)
积分器：    Uo = -1/(RC)∫Ui dt
微分器：    Uo = -RC·dUi/dt
```

**2. 有源滤波器**

| 类型 | 通过频段 | 应用 |
|------|----------|------|
| 低通滤波器 | 0 ~ fc | 去除高频噪声 |
| 高通滤波器 | fc ~ ∞ | 去除直流分量 |
| 带通滤波器 | fc1 ~ fc2 | 选频放大 |
| 带阻滤波器 | 阻fc1~fc2 | 抑制特定频率 |

**3. 比较器**

```
过零比较器：    Uo = +Vsat (Ui>0)
                    -Vsat (Ui<0)

滞回比较器：    具有滞回特性，抗干扰能力强
                UT+ = +(R1/(R1+Rf))·Vsat
                UT- = -(R1/(R1+Rf))·Vsat
```

### 六、实验验证方法

#### 实验1：整流滤波电路验证

**所需器材：**
- 变压器（220V/12V）
- 整流桥（1A）
- 电解电容（470μF/25V）
- 负载电阻（200Ω/5W）
- 万用表、示波器

**测试步骤：**

```
步骤1：测量输入电压
├─ 用万用表测量变压器次级电压（有效值）
└─ 理论值：12V AC

步骤2：测量输出电压
├─ 用万用表测量整流滤波后电压（平均值）
└─ 理论值：Uo ≈ 1.2×12 = 14.4V

步骤3：观察输出波形
├─ 用示波器观察输出波形（应为锯齿波）
├─ 测量纹波电压峰峰值
└─ 测量输出频率（应为100Hz）

步骤4：改变负载
├─ 更换不同阻值的负载电阻
├─ 观察输出电压和纹波的变化
└─ 验证：RL↓ → 纹波↑
```

---

#### 实验2：共射放大电路测试

**静态工作点测量：**

```
1. 断开输入信号和负载
2. 测量各电极电压：
   ├─ UB（基极对地电压）
   ├─ UC（集电极对地电压）
   └─ UE（发射极对地电压）
3. 计算静态电流：
   ├─ IE = UE/RE
   ├─ IC ≈ IE
   └─ IB = IC/β
4. 验证工作区：UCE = UC - UE > 0.7V
```

**动态性能测量：**

```
1. 接入信号源（频率1kHz，幅值10mV）
2. 用示波器同时观察输入输出波形
3. 测量电压放大倍数：
   Au = Uo/Ui（注意相位）
4. 改变输入频率，测试频率响应
5. 增大输入幅值，观察饱和/截止失真
```

---

#### 实验3：功率放大器调试

**消除交越失真步骤：**

```
1. 输入1kHz正弦波
2. 观察输出波形
   ├─ 有交越失真：调整偏置电阻
   ├─ 使输出波形平滑过零
   └─ 测量静态电流（5-10mA为宜）
3. 测量输出功率：Po = Uo²/(2RL)
4. 测量效率：η = Po/PV
```

### 八、典型题型变式

#### 变式1：整流电路变化

**原题型：** 桥式整流+电容滤波

**变式：**
1. **半波整流**：输出频率 = 输入频率（50Hz）
2. **倍压整流**：输出电压 ≈ 2√2Ui
3. **三相整流**：输出频率 = 6倍输入频率（300Hz）

---

#### 变式2：放大电路变化

**原题型：** 分压式偏置共射放大

**变式：**
1. **固定偏置**：直接用R<sub>B</sub>连接VCC，稳定性差
2. **共集放大**：输出从E极取，A<sub>u</sub> ≈ 1
3. **差分放大**：两个管子对称，抑制共模信号
4. **多级放大**：总增益 = 各级增益之积

---

#### 变式3：功率放大变化

**原题型：** OTL互补对称功放

**变式：**
1. **OCL电路**：双电源，无需大电容
2. **准互补电路**：用复合管代替PNP管
3. **甲乙类**：增加偏置，减小交越失真

---

#### 变式4：运放电路变化

**原题型：** 减法器

**变式：**
1. **加法器**：$u_o = -(R_f/R_1 \cdot u_1 + R_f/R_2 \cdot u_2)$
2. **积分器**：$u_o = -\frac{1}{RC}\int u_i dt$
3. **微分器**：$u_o = -RC \frac{du_i}{dt}$
4. **电压比较器**：输出为±V<sub>sat</sub>

### 十、考试答题策略

#### 策略1：读题三遍法

```
第一遍：快速浏览
├─ 了解题目类型和难度
└─ 标记自己有把握的题

第二遍：仔细审题
├─ 圈出关键词（已知、求、要求）
├─ 标注单位
└─ 理解题目物理情景

第三遍：构思解题
├─ 确定解题方法
├─ 列出需要用的公式
└─ 规划答题步骤
```

---

#### 策略2：先易后难

```
答题顺序建议：
1. 填空题（基础，分值高）
2. 简单计算题（套公式即可）
3. 中等难度分析题
4. 综合设计题
5. 最后攻克难题

注意：
- 不要在一道题上耗时过长
- 遇到难题先跳过，做完简单题再回头
- 保证会做的题不失分
```

---

#### 策略3：分步骤得分

```
即使不会做完整道题，也要：
✓ 写出相关公式（公式分）
✓ 代入已知数据（过程分）
✓ 算出中间结果（步骤分）
✓ 画出电路图（图形分）

例：功率放大器题
- 写出Pom = U²om/(2RL) → 得2分
- 算出Uom = VCC - UCES → 得1分
- 即使最终答案错误，也能得3分
```

---

#### 策略4：检查重点

```
优先检查：
1. 单位是否统一（mA→A，kΩ→Ω）
2. 符号是否正确（负号，角度）
3. 公式是否用对（别把共射公式用到共集）
4. 数值是否合理（用数量级估算）
5. 必要步骤是否完整

时间不够时：
只检查分值最大的题目
```
