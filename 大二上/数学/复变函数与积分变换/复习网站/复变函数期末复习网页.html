📐 复数运算类 (12分)

3题 × 4分
题型1: 辐角计算
例题: 设 z = (1-√3i)(1+i) / [(-2+2i)(1-i)]，则 arg z = ______
⭐⭐⭐ 中等 辐角性质 分子分母法 4分

解题技巧

方法一: 利用 arg(z₁z₂) = arg z₁ + arg z₂
方法二: 直接计算化简为 a+bi 形式，判断象限

题型2: 复数的幂
例题: 设 z = i^i，则 z 的代数形式为 ______
⭐⭐⭐⭐ 较难 复数幂运算 指数定义法 4分

核心公式

a^b = e^(b·ln a)
ln i = i·π/2 (主值)
i^i = e^(-π/2) ≈ 0.2079

易错点

复数的幂运算结果可能是实数！不要想当然认为一定是复数形式。

题型3: 根式计算
例题: 根式 ∛[-8(1+i)] = ______
⭐⭐⭐⭐ 较难 复数根式 指数形式展开 4分

解题步骤

1. 化为指数形式 z = r·e^(iθ)
2. 应用公式 ∛z = ∛r · e^(i(θ+2kπ)/3), k=0,1,2
3. 写出所有三个根

🔍 复积分类 (8分)

2题 × 4分
题型1: 柯西定理应用
例题: ∮_{|z|=1} sin(2z+1)/(z-2)² dz = ______
⭐⭐ 简单 柯西定理 奇点判断 4分

关键判断

1. 找出奇点位置: z = 2
2. 判断奇点是否在围道内: |2| = 2 > 1 (在外部)
3. 函数在围道内解析 → 积分为 0

题型2: 原函数法
例题: ∫₀^(1+i) (3z + 2)dz = ______
⭐ 容易 解析函数积分 牛顿-莱布尼茨公式 4分

解题要点

F(z) = (3z²/2) + 2z
积分 = F(1+i) - F(0)

📊 级数与留数类 (16分)

4题 × 4分
题型1: 奇点类型判定
例题: z = 0 是函数 (sin z - z)/z² 的 ______ 奇点
⭐⭐⭐ 中等 奇点分类 泰勒展开 4分

判定方法

可去奇点: 洛朗展开无负幂项
极点: 有限个负幂项
本性奇点: 无限个负幂项

题型2: 级数收敛范围
例题: 级数 Σ(n+1)(z+1)ⁿ/2ⁿ 的收敛范围为 ______
⭐⭐⭐ 中等 幂级数收敛 比值判别法 4分
题型3: 留数计算
例题: Res[1/(z(e^z-1)), 0] = ______
⭐⭐⭐⭐ 较难 留数计算 洛朗级数 4分

核心步骤

1. e^z - 1 = z(1 + z/2 + z²/6 + ...)
2. 展开为洛朗级数
3. 找 z^(-1) 项系数
题型4: 留数定理积分
例题: ∮_{|z|=1} (z²+1)/z⁴ dz = ______
⭐⭐ 简单 留数定理 展开观察 4分

关键提示

展开后没有 z^(-1) 项时，留数 = 0，积分 = 0

题型3: 幂级数收敛性
例题: 如果级数 Σcₙzⁿ 在 z=1+2i 点收敛，则级数在 ( )
A. z=1+2i 点条件收敛

📝 计算题专项训练 (52分)

📈 罗朗级数展开 (10分)

必考题型
典型题: 求函数 f(z) = 1/[z²(2+z)] 在 2 < |z| < ∞ 内展为罗朗级数
⭐⭐⭐⭐ 较难 罗朗展开 部分分式法 10分

解题流程

步骤1: 部分分式分解
步骤2: 根据区域选择展开方式 (|z|>2 时用 1/(1+2/z))
步骤3: 应用几何级数展开
步骤4: 合并同类项

核心技巧

|z| > a 时: 1/(a+z) = (1/z)·1/(1+a/z)
= (1/z)·Σ(-a/z)ⁿ = Σ(-1)ⁿaⁿ/z^(n+1)

🎯 泰勒级数展开 (10分)

必考题型
典型题: 设 f(z) = (z-1)/[(z-2)(2z+1)]，求 f(z) 在 z₀=0 处的泰勒展开式，并指出收敛半径
⭐⭐⭐⭐ 较难 泰勒展开 部分分式法 10分

解题要点

收敛半径: 等于到最近奇点的距离
展开方式: 部分分式 → 几何级数
注意: 取两个展开式的公共收敛域

🔄 求共轭调和函数 (10分)

必考题型
典型题: 已知调和函数 u(x,y) = 2xy - 3y - 2x，求函数 v(x,y)，使 f(z)=u+iv 为解析函数，且满足 f(0)=1
⭐⭐⭐ 中等 C-R方程 积分法 10分

C-R方程组

∂v/∂y = ∂u/∂x
∂v/∂x = -∂u/∂y

解题步骤

1. 验证 u 是调和函数 (∇²u = 0)
2. 利用 C-R 方程求 v 的偏导数
3. 对其中一个方程积分得 v
4. 代入另一个方程确定待定函数
5. 利用初值条件确定常数

⭕ 留数定理计算积分 (10分)

必考题型
典型题: 利用留数定理计算 ∮_C cos z/[(z+2)²(z-1)] dz，其中 C 是正向圆周 |z|=3
⭐⭐⭐⭐ 较难 留数定理 留数公式 10分

留数计算公式

一阶极点: Res[f,z₀] = lim(z→z₀) (z-z₀)f(z)
二阶极点: Res[f,z₀] = lim(z→z₀) d/dz[(z-z₀)²f(z)]
留数定理: ∮f(z)dz = 2πi·ΣRes[f,zₖ]

解题步骤

1. 确定围道内的所有奇点
2. 判断每个奇点的类型和阶数
3. 用相应公式计算各奇点的留数
4. 应用留数定理求和

🌊 傅里叶变换 (12分)

必考题型
典型题: 已知函数 f(t) = (t+1)e^(-2t)·u(t)，求 f(t) 的傅里叶变换及其积分表达式
⭐⭐⭐⭐ 较难 傅里叶变换 分部积分 12分

傅里叶变换定义

F(ω) = ∫_{-∞}^{∞} f(t)e^(-iωt) dt
f(t) = (1/2π)∫_{-∞}^{∞} F(ω)e^(iωt) dω

解题技巧

分部积分法:
∫t·e^(-αt)dt 令 u=t, dv=e^(-αt)dt
结果化简:
注意合并同类项，整理为标准形式

📖 核心公式速查手册

复数基本公式

z = x + iy = r·e^(iθ)
|z| = √(x²+y²)
arg z = arctan(y/x)
z^w = e^(w·ln z)
ln z = ln|z| + i(arg z + 2kπ)

C-R方程

∂u/∂x = ∂v/∂y
∂u/∂y = -∂v/∂x

f'(z) = ∂u/∂x + i·∂v/∂x
f'(z) = ∂v/∂y - i·∂u/∂y

柯西积分公式

∮_C f(z)dz = 0 (f解析)

f(z₀) = (1/2πi)∮_C f(z)/(z-z₀) dz

f^(n)(z₀) = (n!/2πi)∮_C f(z)/(z-z₀)^(n+1) dz

常用级数展开

e^z = 1 + z + z²/2! + z³/3! + ...
sin z = z - z³/3! + z⁵/5! - ...
cos z = 1 - z²/2! + z⁴/4! - ...
1/(1-z) = 1 + z + z² + ..., |z|<1
1/(1+z) = 1 - z + z² - ..., |z|<1

留数计算

∮_C f(z)dz = 2πi·ΣRes[f,zₖ]

一阶极点:
Res[f,z₀] = lim(z→z₀)(z-z₀)f(z)

m阶极点:
Res[f,z₀] = [1/(m-1)!]·lim(z→z₀)
d^(m-1)/dz^(m-1)[(z-z₀)^m f(z)]

傅里叶变换

F(ω) = ∫_{-∞}^{∞} f(t)e^(-iωt) dt

f(t) = (1/2π)∫_{-∞}^{∞} F(ω)e^(iωt) dω

F[e^(-at)u(t)] = 1/(a+iω)
F[te^(-at)u(t)] = 1/(a+iω)²

⚠️ 考试易错点全梳理

复数运算易错点

  • 辐角计算忘记判断象限，尤其是第三、四象限
  • i^i = e^(-π/2) 是实数，不是复数！
  • 复数的 n 次方根有 n 个值，不要漏掉
  • ln z 是多值函数: ln z = ln|z| + i(arg z + 2kπ)

解析函数易错点

  • C-R方程第二式有负号: ∂u/∂y = -∂v/∂x
  • 调和函数的实部和虚部可互换，但反之不一定
  • e^z 整个平面解析，但 e^(z̄) 处处不解析
  • 解析必连续，但连续不一定解析

复积分易错点

  • 柯西定理仅对单连通区域内的解析函数成立
  • 计算前必须先判断奇点是否在围道内
  • 积分方向：正向 = 逆时针
  • 奇点在围道外 → 积分为 0

级数展开易错点

  • 泰勒级数收敛半径 = 到最近奇点的距离
  • 罗朗级数的展开方式取决于区域
  • |z| > a 时: 1/(a+z) = (1/z)·1/(1+a/z)
  • |z| < a 时: 1/(a+z) = (1/a)·1/(1+z/a)
  • 部分分式分解时系数计算容易出错

留数计算易错点

  • 可去奇点的留数为 0，容易被遗漏
  • 不同阶的极点用不同的留数公式
  • 二阶极点求导后要记得化简
  • 只有 z^(-1) 项对积分有贡献

函数性质易错点

  • 复变函数 sin z, cos z 是无界的！
  • 实函数的有界性在复平面不成立
  • Ln z 是多值函数，ln z 是主值
  • 周期性在复平面仍然成立

考试答题技巧

  • 时间分配: 填空30分钟，选择10分钟，计算50分钟
  • 先易后难: 先做有把握的题目
  • 检查奇点: 积分和级数题先确定奇点位置
  • 部分分式: 有理函数优先考虑分解
  • 保留步骤: 即使最终答案错误也能得分
  • 验证结果: 利用特殊值检验答案合理性
