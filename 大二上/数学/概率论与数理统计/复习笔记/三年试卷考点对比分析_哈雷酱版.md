## 💡 七、重点公式汇总（按考频排序）

### 🥇 必背公式（三年必考）

#### 1. 全概率公式与贝叶斯公式
```
全概率公式：P(B) = Σ P(Aᵢ)P(B|Aᵢ)
贝叶斯公式：P(Aᵢ|B) = P(Aᵢ)P(B|Aᵢ) / P(B)
```
**考法：** 实际问题建模（飞机搜索、求职、生产合格率等）

---

#### 2. 随机变量函数的密度（单调变换）
```
Y = g(X)，g单调，则：
fᵧ(y) = fₓ(h(y))·|h'(y)|

特别地，Y = aX + b：
fᵧ(y) = (1/|a|)·fₓ((y-b)/a)
```
**考法：** 三年全考线性变换 Y = aX + b

---

#### 3. 二维随机变量
```
边缘密度：fₓ(x) = ∫_{-∞}^{+∞} f(x,y)dy
          fᵧ(y) = ∫_{-∞}^{+∞} f(x,y)dx

数学期望：E(XY) = ∫∫ xy·f(x,y)dxdy
         E(g(X,Y)) = ∫∫ g(x,y)·f(x,y)dxdy
```
**考法：** 给定联合密度，求边缘密度和E(XY)或E(Y²)

---

#### 4. 中心极限定理（独立同分布）
```
X₁,X₂,...,Xₙ i.i.d., E(Xᵢ)=μ, D(Xᵢ)=σ²

则：ΣXᵢ 近似~ N(nμ, nσ²)

标准化：(ΣXᵢ - nμ)/(σ√n) 近似~ N(0,1)
```
**考法：** 实际问题应用（用电、索赔、等车等），用Φ表示

---

#### 5. 矩估计
```
原理：用样本矩代替总体矩
方法：
  1. 求E(X)，用参数θ表示
  2. 令E(X) = X̄
  3. 解出θ̂ = g(X̄)
```
**考法：** 给定含参数θ的密度函数，求θ的矩估计量

---

#### 6. 正态总体均值的假设检验（σ²已知）
```
H₀: μ = μ₀  vs  H₁: μ ≠ μ₀

检验统计量：Z = (X̄ - μ₀)/(σ/√n) ~ N(0,1)

拒绝域（α=0.05）：|Z| > Z_{α/2} = 1.96
```
**考法：** 给定样本数据，检验均值是否有显著变化

---

### 🥈 高频公式（两年考）

#### 7. 条件概率
```
P(B|A) = P(AB) / P(A)
P(AB) = P(B|A)·P(A)
P(A∪B) = P(A) + P(B) - P(AB)
```
**考法：** 填空题或选择题，考察条件概率的基本计算

---

#### 8. 分布函数/密度函数的凸组合
```
若F₁(x), F₂(x)是分布函数，则：
F(x) = αF₁(x) + βF₂(x) 是分布函数

⇔ α + β = 1, α ≥ 0, β ≥ 0

同样适用于密度函数！
```
**考法：** 选择题，求系数α和β（23-24、24-25连考）

---

#### 9. 正态分布的线性组合
```
X ~ N(μ₁,σ₁²), Y ~ N(μ₂,σ₂²) 独立

则：aX + bY ~ N(aμ₁ + bμ₂, a²σ₁² + b²σ₂²)
```
**考法：** 选择题，求X±Y或aX+bY的分布（三年必考）

---

#### 10. 无偏估计
```
定义：E(θ̂) = θ

判断方法：计算估计量的期望，验证是否等于参数

有效性：在所有无偏估计中，方差最小的最有效
```
**考法：** 选择题，判断哪个是/不是无偏估计（三年必考）

### 🥉 常规公式

#### 11. 常用分布的数字特征

| 分布 | 记号 | E(X) | D(X) |
|------|------|------|------|
| 二项分布 | B(n,p) | np | np(1-p) |
| 泊松分布 | P(λ) | λ | λ |
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)²/12 |
| 指数分布 | Exp(λ) | 1/λ | 1/λ² |
| 正态分布 | N(μ,σ²) | μ | σ² |

---

#### 12. 方差的性质
```
D(aX+b) = a²D(X)
D(X±Y) = D(X) + D(Y)  （X,Y独立）
D(X) = E(X²) - [E(X)]²
```

---

#### 13. 抽样分布

**χ²分布：**
```
X₁,...,Xₙ ~ N(μ,σ²) i.i.d.

则：(n-1)S²/σ² ~ χ²(n-1)
   或：Σ(Xᵢ-μ)²/σ² ~ χ²(n)
```

**t分布：**
```
X̄ ~ N(μ, σ²/n), S是样本标准差

则：(X̄ - μ)/(S/√n) ~ t(n-1)
```

---

#### 14. 契比雪夫不等式
```
P{|X - E(X)| ≥ ε} ≤ D(X)/ε²

或：P{|X - E(X)| < ε} ≥ 1 - D(X)/ε²
```

### 📖 重点题型专项训练

#### 专项1：全概率公式（必考10分）

**标准套路：**
1. 识别"原因"事件A₁, A₂, ..., Aₙ（完备事件组）
2. 列出P(Aᵢ)和P(B|Aᵢ)
3. 套公式：P(B) = ΣP(Aᵢ)P(B|Aᵢ)

**训练建议：** 做5道不同背景的应用题（搜索、生产、医疗等）

---

#### 专项2：随机变量函数密度（必考10分）

**标准套路（Y=aX+b）：**
1. 求反函数：x = h(y) = (y-b)/a
2. 求导数：h'(y) = 1/a
3. 套公式：fᵧ(y) = fₓ(h(y))·|h'(y)|
4. **确定值域！** x∈(c,d) ⇒ y∈(ac+b, ad+b)

**训练建议：** 做10道线性变换题，熟练到闭眼都会！

---

#### 专项3：二维随机变量（必考10分）

**标准套路：**
1. 画出定义域（矩形或三角形）
2. 求边缘密度：
   - fₓ(x)：对y积分（确定y的范围！）
   - fᵧ(y)：对x积分（确定x的范围！）
3. 求E(XY)：二重积分∫∫xyf(x,y)dxdy

**训练建议：** 做5道不同定义域的题（矩形、三角、圆等）

---

#### 专项4：中心极限定理（必考10分）

**标准套路：**
1. 设X = ΣXᵢ
2. 算E(X) = nμ, D(X) = nσ²
3. 标准化：P{a ≤ X ≤ b} = P{(a-nμ)/(σ√n) ≤ Z ≤ (b-nμ)/(σ√n)}
4. 用Φ表示：Φ(上限) - Φ(下限)

**训练建议：** 做5道实际应用题，练习标准化

---

#### 专项5：矩估计（必考10分）

**标准套路：**
1. 写出E(X) = ∫xf(x;θ)dx（用θ表示）
2. 令E(X) = X̄
3. 解方程得θ̂ = g(X̄)

**训练建议：** 做5道不同密度函数的题

---

#### 专项6：假设检验（必考10分）

**五步法（必须写全！）：**
1. 建立假设：H₀: μ=μ₀, H₁: μ≠μ₀
2. 选择统计量：Z = (X̄-μ₀)/(σ/√n)
3. 确定拒绝域：|Z| > Z_{α/2}
4. 计算统计量的值：代入数据算出Z
5. 做出判断：比较Z与临界值，下结论

**训练建议：** 做5道题，练到步骤默写无误

### 💎 终极秘籍：考前必背清单

#### 公式清单（A4纸一张）

**第一部分：概率论基础**
```
1. P(A∪B) = P(A) + P(B) - P(AB)
2. P(B|A) = P(AB)/P(A)
3. 全概率：P(B) = ΣP(Aᵢ)P(B|Aᵢ)
4. 独立：P(AB) = P(A)P(B)
```

**第二部分：常用分布**
```
         E(X)      D(X)
B(n,p):  np       np(1-p)
P(λ):    λ        λ
U(a,b):  (a+b)/2  (b-a)²/12
N(μ,σ²): μ        σ²
```

**第三部分：核心公式**
```
1. Y=aX+b: fᵧ(y) = (1/|a|)fₓ((y-b)/a)
2. 边缘密度：fₓ(x) = ∫f(x,y)dy
3. X+Y ~ N(μ₁+μ₂, σ₁²+σ²₂)  [独立正态]
4. D(aX+b) = a²D(X)
5. 中心极限：ΣXᵢ ~ N(nμ, nσ²)
6. χ²(n-1) = (n-1)S²/σ²
7. t(n-1) = (X̄-μ)/(S/√n)
8. Z检验拒绝域：|Z| > 1.96 (α=0.05)
```

#### 易错点清单

| 序号 | 易错点 | 正确做法 |
|------|--------|----------|
| 1 | N(5,9)是什么分布？ | μ=5, **σ²=9**，不是σ=9！ |
| 2 | Y=aX+b的值域 | x∈(c,d) ⇒ y∈(ac+b,ad+b) |
| 3 | 边缘密度积分限 | 画图确定另一个变量的范围！ |
| 4 | χ²/t分布自由度 | 都是n-1，不是n！ |
| 5 | 中心极限定理分母 | σ√n，不是σ²√n！ |
| 6 | 矩估计 | E(X)=X̄，不是E(X²)=X̄²！ |
| 7 | 假设检验结论 | 接受H₀≠证明H₀正确！ |
