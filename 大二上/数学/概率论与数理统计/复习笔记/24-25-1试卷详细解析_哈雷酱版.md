## 🎯 一、填空题（共20分）

### 第1题：古典概型 - 无放回抽样问题

**题目：** 10张奖券中含有4张中奖的奖券，每人购买1张，则第二人中奖的概率为___。

**答案：** `2/5`

**📖 考点分析：**
- **核心考点：** 古典概型、对称性原理
- **难度：** ⭐⭐☆☆☆

**💡 详细解析：**

哼，这道题很多笨蛋会被"第二人"这个条件吓到，其实根本不用慌！(￣▽￣)

**方法一：对称性原理（最优雅的解法！）**

本小姐告诉你一个高贵的原理：在无放回抽样中，**每个人抽到中奖券的概率都是相同的**！

这是因为：
- 第一人抽奖时，中奖概率 = 4/10
- 第二人抽奖时，虽然第一人已经抽走一张，但**第二人不知道第一人的结果**
- 从第二人的角度看，他面对的仍然是"10张券中有4张中奖"的局面

所以第二人中奖概率 = **4/10 = 2/5**

**方法二：全概率公式（适合较真的笨蛋）**

设A₁表示"第一人中奖"，A₂表示"第一人不中奖"
设B表示"第二人中奖"

则：
```
P(B) = P(A₁)P(B|A₁) + P(A₂)P(B|A₂)
     = (4/10) × (3/9) + (6/10) × (4/9)
     = 12/90 + 24/90
     = 36/90
     = 2/5
```

**🎓 知识拓展：**
> 推广到一般情形：N个物品中有M个特殊品，无放回抽取，则**任何一次**抽到特殊品的概率都是 M/N

### 第2题：条件概率与概率加法公式

**题目：** 已知P(A) = 0.4, P(B) = 0.3, P(B|A) = 0.4，则P(A∪B) = ___。

**答案：** `0.66`

**📖 考点分析：**
- **核心考点：** 条件概率、概率加法公式、事件的交与并
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

笨蛋，这道题考察的是条件概率和并事件概率的关系！本小姐来给你理清楚～(￣ω￣)

**解题步骤：**

**Step 1：** 利用条件概率定义求P(AB)

条件概率定义：
```
P(B|A) = P(AB) / P(A)
```

因此：
```
P(AB) = P(B|A) × P(A)
      = 0.4 × 0.4
      = 0.16
```

**Step 2：** 利用概率加法公式求P(A∪B)

```
P(A∪B) = P(A) + P(B) - P(AB)
        = 0.4 + 0.3 - 0.16
        = 0.66
```

**🔑 关键公式：**
1. **条件概率：** P(B|A) = P(AB) / P(A)
2. **加法公式：** P(A∪B) = P(A) + P(B) - P(AB)

**⚠️ 常见错误：**
- 直接加P(A) + P(B) = 0.7（忘记减去交集部分）
- 混淆P(B|A)和P(AB)的含义

**🎓 知识点总结：**
- P(AB)表示A和B同时发生的概率
- P(B|A)表示在A发生的条件下B发生的概率
- 两者关系：P(AB) = P(B|A) × P(A) = P(A|B) × P(B)

### 第3题：二项分布

**题目：** 设X ~ B(2, p)，已知P(X ≥ 1) = 5/9，则p = ___。

**答案：** `1/3`

**📖 考点分析：**
- **核心考点：** 二项分布、概率分布列、对立事件
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

哼，二项分布是本小姐最喜欢的分布之一呢！(￣▽￣)ノ

**知识回顾：**

二项分布X ~ B(n, p)的概率分布：
```
P(X = k) = C(n,k) × p^k × (1-p)^(n-k), k = 0,1,2,...,n
```

本题中 n = 2，所以：
- P(X = 0) = C(2,0) × p⁰ × (1-p)² = (1-p)²
- P(X = 1) = C(2,1) × p¹ × (1-p)¹ = 2p(1-p)
- P(X = 2) = C(2,2) × p² × (1-p)⁰ = p²

**解题步骤：**

**方法一：利用对立事件（最简洁！）**

```
P(X ≥ 1) = 1 - P(X = 0)
5/9 = 1 - (1-p)²
(1-p)² = 1 - 5/9 = 4/9
1-p = 2/3  （取正值，因为0 < p < 1）
p = 1/3
```

**方法二：直接展开（适合保险起见）**

```
P(X ≥ 1) = P(X = 1) + P(X = 2)
         = 2p(1-p) + p²
         = 2p - 2p² + p²
         = 2p - p²
         = 5/9

即：9p² - 18p + 5 = 0
解得：p = 1/3 或 p = 5/3

因为 0 < p < 1，所以 p = 1/3
```

**🎓 解题技巧：**
- 遇到"至少"、"至多"类问题，优先考虑**对立事件**
- P(X ≥ k) = 1 - P(X < k) 通常更简单
- 二项分布中，P(X = 0) = (1-p)ⁿ 是最简单的项

### 第4题：概率密度函数的归一化

**题目：** 已知X的密度函数f(x) = ce^(-|x|), -∞ < x < +∞，则c = ___。

**答案：** `1/2`

**📖 考点分析：**
- **核心考点：** 概率密度函数的性质、积分计算、绝对值函数
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

笨蛋，概率密度函数最重要的性质就是**归一化条件**！(￣へ￣)

**核心性质：**
```
∫_{-∞}^{+∞} f(x)dx = 1
```

**解题步骤：**

**Step 1：** 处理绝对值函数

```
e^(-|x|) = { e^x,    x < 0
           { e^(-x), x ≥ 0
```

**Step 2：** 分段积分

```
∫_{-∞}^{+∞} ce^(-|x|)dx = c[∫_{-∞}^0 e^x dx + ∫_0^{+∞} e^(-x) dx]
```

**Step 3：** 计算各部分

```
∫_{-∞}^0 e^x dx = e^x |_{-∞}^0 = e^0 - 0 = 1

∫_0^{+∞} e^(-x) dx = -e^(-x) |_0^{+∞} = 0 - (-1) = 1
```

**Step 4：** 应用归一化条件

```
c(1 + 1) = 1
2c = 1
c = 1/2
```

**🔑 关键技巧：**
1. 遇到绝对值，按0点分段
2. 注意积分上下限，尤其是无穷积分
3. e^(-∞) = 0, e^(+∞) = +∞ 要牢记

**🎓 拓展知识：**
> 这个密度函数f(x) = (1/2)e^(-|x|)称为**拉普拉斯分布**（Laplace Distribution），也叫双指数分布，在信号处理和机器学习中很常见！

### 第5题：抽样分布 - t分布

**题目：** 设X₁, X₂, ⋯, X₁₆是来自N(0,4)的样本，S是其样本标准差，则统计量 T = (1/16)Σ(Xᵢ/4S) 服从___分布。

**答案：** `t(15)分布`

**📖 考点分析：**
- **核心考点：** 正态总体的抽样分布、t分布的构造、标准化
- **难度：** ⭐⭐⭐⭐☆

**💡 详细解析：**

哼，这道题是抽样分布的经典题型，考察你对t分布本质的理解！(￣ω￣)

**知识储备：**

已知：X₁, X₂, ..., X₁₆ ~ N(0, 4)，即μ = 0, σ² = 4, σ = 2

**关键结论：**
1. 样本均值：X̄ ~ N(0, 4/16) = N(0, 1/4)
2. 标准化后：X̄/(1/2) ~ N(0, 1)
3. (n-1)S²/σ² ~ χ²(n-1)，即 15S²/4 ~ χ²(15)

**解题步骤：**

**Step 1：** 化简统计量

```
T = (1/16)Σ(Xᵢ/4S)
  = (1/16) × (1/4S) × ΣXᵢ
  = (1/64S) × 16X̄
  = X̄/(4S)
```

**Step 2：** 标准化分子

```
X̄ ~ N(0, σ²/n) = N(0, 4/16) = N(0, 1/4)

所以：X̄/(1/2) ~ N(0, 1)

因此：X̄ = (1/2) × [X̄/(1/2)]
```

**Step 3：** 改写统计量

```
T = X̄/(4S)
  = [X̄/(1/2)] / [4S/(1/2)]
  = [X̄/(1/2)] / (8S)
```

等等，让本小姐重新整理一下...

**正确推导：**

```
T = X̄/(4S)

标准化：X̄/(σ/√n) = X̄/(2/4) = 2X̄ ~ N(0,1)

所以：X̄ = (2X̄)/2

T = X̄/(4S) = (2X̄)/(8S) = [2X̄] / [8S]
```

由于：
- 2X̄ ~ N(0, 1)
- (n-1)S²/σ² = 15S²/4 ~ χ²(15)
- 所以 S/σ = S/2 = √[χ²(15)/15]

最终：
```
T = [标准正态] / [√(χ²(15)/15)]
```

根据t分布的定义：
```
t(n) = Z / √(χ²(n)/n)，其中Z ~ N(0,1)
```

**因此 T ~ t(15)**

**🔑 t分布的标准构造：**
```
t(n) = [N(0,1)] / √[χ²(n)/n]
```

记住：自由度 = 样本量 - 1 = 16 - 1 = 15

### 第1题：事件的独立性

**题目：** 设事件A, B相互独立，C = ∅，则A, B, C ( )
(A) 相互独立
(B) 两两独立，不相互独立
(C) 不一定两两独立
(D) 仅两两独立

**答案：** `A`

**📖 考点分析：**
- **核心考点：** 事件独立性的定义、空集的性质
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

哼，这道题考察的是独立性的本质定义！笨蛋要分清楚**两两独立**和**相互独立**的区别哦～(￣▽￣)

**知识回顾：**

**两两独立：** P(AB) = P(A)P(B), P(AC) = P(A)P(C), P(BC) = P(B)P(C)

**相互独立：** 除了两两独立，还需满足
```
P(ABC) = P(A)P(B)P(C)
```

**空集的性质：**
- P(∅) = 0
- P(A∅) = 0
- ∅与任何事件A都独立，因为 P(A∅) = 0 = P(A) × 0 = P(A) × P(∅)

**验证步骤：**

**1. 验证两两独立：**
```
P(AB) = P(A)P(B) ✓ （已知A, B独立）
P(AC) = P(A∅) = 0 = P(A) × 0 = P(A)P(C) ✓
P(BC) = P(B∅) = 0 = P(B) × 0 = P(B)P(C) ✓
```

**2. 验证相互独立：**
```
P(ABC) = P(AB∅) = P(∅) = 0
P(A)P(B)P(C) = P(A) × P(B) × 0 = 0
所以 P(ABC) = P(A)P(B)P(C) ✓
```

**结论：** A, B, C **相互独立**！

**🎓 重要结论：**
> 空集∅与任何事件都独立！这是空集的一个重要性质。

### 第2题：分布函数的性质

**题目：** 设函数F₁(x)和F₂(x)分别为随机变量X₁和X₂的分布函数，若F(x) = aF₁(x) - bF₂(x)是某随机变量的分布函数，则 ( )
(A) a = -3/2, b = 1/2
(B) a = 3/2, b = -1/2
(C) a = 2/3, b = -1/3
(D) a = 2, b = -1

**答案：** `C`

**📖 考点分析：**
- **核心考点：** 分布函数的性质、非负性、归一性
- **难度：** ⭐⭐⭐⭐☆

**💡 详细解析：**

笨蛋，分布函数可不是随便一个函数都能当的！本小姐来告诉你它必须满足的条件～(￣ω￣)

**分布函数的三大性质：**
1. **单调非减性：** F(x)单调不减
2. **归一性：** lim(x→-∞) F(x) = 0, lim(x→+∞) F(x) = 1
3. **右连续性：** F(x+0) = F(x)

**解题关键：** 利用归一性条件

**Step 1：** 在x → -∞时
```
F(-∞) = aF₁(-∞) - bF₂(-∞) = a×0 - b×0 = 0 ✓
```
自动满足！

**Step 2：** 在x → +∞时
```
F(+∞) = aF₁(+∞) - bF₂(+∞) = a×1 - b×1 = a - b = 1
```
得到第一个方程：**a - b = 1** ... ①

**Step 3：** 非负性条件

由于F(x)是分布函数，必须满足 0 ≤ F(x) ≤ 1 对所有x成立。

注意到如果b < 0（即-b > 0），则：
```
F(x) = aF₁(x) + (-b)F₂(x)
```
这是两个分布函数的**凸组合**，要保证非负性，需要：
- a ≥ 0
- -b ≥ 0，即 b ≤ 0
- a + (-b) = 1，即 a - b = 1

从选项来看，满足a - b = 1且a, -b > 0的只有：
- (C) a = 2/3, b = -1/3：2/3 - (-1/3) = 2/3 + 1/3 = 1 ✓
- (D) a = 2, b = -1：2 - (-1) = 3 ✗

但D选项中a + (-b) = 2 + 1 = 3 ≠ 1，不能保证F(x) ≤ 1

**验证C选项：**
```
F(x) = (2/3)F₁(x) + (1/3)F₂(x)
```
这是两个分布函数的凸组合（系数和为1，都非负），必然是分布函数！

**答案：C**

**🎓 知识点总结：**
> 若F(x) = αF₁(x) + βF₂(x)是分布函数，则需满足：
> - α + β = 1（归一性）
> - α ≥ 0, β ≥ 0（非负性）
>
> 这称为分布函数的**凸组合**！

### 第3题：方差的性质

**题目：** 设随机变量X和Y相互独立，X ~ U(1,7), Y ~ N(5,9)，则D(3X - Y + 2) = ( )
(A) 18
(B) 36
(C) 38
(D) 20

**答案：** `B`

**📖 考点分析：**
- **核心考点：** 方差的性质、均匀分布与正态分布的方差、独立性
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

哼，这道题考察方差的计算！笨蛋们要注意看清楚分布类型哦～(￣▽￣)ノ

**⚠️ 审题关键：**
- X ~ **U**(1,7) 是**均匀分布**
- Y ~ **N**(5,9) 是**正态分布**

**知识储备：**

**1. 均匀分布U(a,b)的方差：**
```
D(X) = (b-a)²/12
```

**2. 正态分布N(μ,σ²)的方差：**
```
D(Y) = σ²  （第二个参数直接就是方差！）
```

**3. 方差的性质：**
- D(aX + b) = a²D(X)（常数不影响方差）
- D(X ± Y) = D(X) + D(Y)（X, Y独立时）
- D(aX + bY) = a²D(X) + b²D(Y)（X, Y独立时）

**解题步骤：**

**Step 1：** 计算D(X)
```
X ~ U(1, 7)（均匀分布）
D(X) = (7-1)²/12 = 36/12 = 3
```

**Step 2：** 计算D(Y)
```
Y ~ N(5, 9)（正态分布）
D(Y) = 9  ← 正态分布的第二个参数直接是方差！
```

**Step 3：** 计算D(3X - Y + 2)

由于X, Y独立，且常数2不影响方差：
```
D(3X - Y + 2) = D(3X) + D(-Y)
              = 3²D(X) + (-1)²D(Y)
              = 9D(X) + D(Y)
              = 9×3 + 9
              = 27 + 9
              = 36
```

**答案：B (36)**

**🔑 方差计算口诀：**
- 系数平方提出来
- 加减变成加（独立时）
- 常数方差为零

**⚠️ 本题易错点：**
1. **混淆分布类型：** X是均匀分布，Y是正态分布，计算方差的公式不同！
2. **正态分布参数：** N(μ,σ²)中，第二个参数σ²**直接就是方差**，不需要再计算
3. **均匀分布公式：** U(a,b)的方差是(b-a)²/12，别忘了除以12

**🎓 分布对比总结：**

| 分布 | 记号 | 期望E(X) | 方差D(X) |
|------|------|----------|----------|
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)²/12 |
| 正态分布 | N(μ,σ²) | μ | σ² |

### 第4题：正态分布的线性组合

**题目：** 设X ~ N(0,1), Y ~ N(1,1), X和Y相互独立，则( )
(A) P{X + Y ≤ 0} = 1/2
(B) P{X + Y ≤ 1} = 1/2
(C) P{X - Y ≤ 0} = 1/2
(D) P{X - Y ≤ 1} = 1/2

**答案：** `B`

**📖 考点分析：**
- **核心考点：** 正态分布的线性组合性质、标准正态分布
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

哼，正态分布的线性组合还是正态分布，这是本小姐最喜欢的性质！(￣ω￣)

**核心性质：**

若X ~ N(μ₁, σ₁²), Y ~ N(μ₂, σ₂²)，且X, Y独立，则：
```
aX + bY ~ N(aμ₁ + bμ₂, a²σ₁² + b²σ₂²)
```

**已知条件：**
- X ~ N(0, 1)：μₓ = 0, σₓ² = 1
- Y ~ N(1, 1)：μᵧ = 1, σᵧ² = 1

**逐项检验：**

**选项A：P{X + Y ≤ 0}**
```
X + Y ~ N(0+1, 1+1) = N(1, 2)

标准化：(X+Y-1)/√2 ~ N(0,1)

P{X + Y ≤ 0} = P{(X+Y-1)/√2 ≤ (0-1)/√2}
              = P{Z ≤ -1/√2}
              = Φ(-0.707)
              < 0.5  ✗
```

**选项B：P{X + Y ≤ 1}**
```
X + Y ~ N(1, 2)

P{X + Y ≤ 1} = P{(X+Y-1)/√2 ≤ (1-1)/√2}
              = P{(X+Y-1)/√2 ≤ 0}
              = Φ(0)
              = 0.5  ✓
```

**选项C：P{X - Y ≤ 0}**
```
X - Y ~ N(0-1, 1+1) = N(-1, 2)

P{X - Y ≤ 0} = P{(X-Y+1)/√2 ≤ 1/√2}
              = Φ(0.707)
              > 0.5  ✗
```

**选项D：P{X - Y ≤ 1}**
```
X - Y ~ N(-1, 2)

P{X - Y ≤ 1} = P{(X-Y+1)/√2 ≤ 2/√2}
              = P{Z ≤ √2}
              = Φ(1.414)
              > 0.5  ✗
```

**答案：B**

**🎓 解题技巧：**
> 对于正态分布Z ~ N(μ, σ²)：
> - P{Z ≤ μ} = 0.5（均值处的概率为0.5）
> - 这是正态分布对称性的体现！

### 第5题：无偏估计的有效性

**题目：** 设X₁, X₂, X₃是来自总体X ~ N(μ, σ²)的简单随机样本，则下列μ的无偏估计中，方差最小的是( )
(A) (1/2)X₁ + (1/2)X₂
(B) (1/4)X₁ + (3/4)X₂
(C) (2/3)X₁ + (1/3)X₂
(D) (1/3)X₁ + (1/3)X₂ + (1/3)X₃

**答案：** `D`

**📖 考点分析：**
- **核心考点：** 无偏估计、估计量的有效性、方差比较
- **难度：** ⭐⭐⭐⭐☆

**💡 详细解析：**

哼，这道题考察的是估计量的**有效性**问题！笨蛋要记住，无偏只是基本要求，方差小才是王道！(￣▽￣)／

**知识储备：**

**1. 无偏性：** E(估计量) = 真实参数
**2. 有效性：** 在所有无偏估计中，方差最小的称为**最有效估计**

**验证无偏性：**

对于形如 θ̂ = a₁X₁ + a₂X₂ + a₃X₃ 的估计量：
```
E(θ̂) = a₁E(X₁) + a₂E(X₂) + a₃E(X₃)
      = (a₁ + a₂ + a₃)μ
```

要无偏，需满足：**a₁ + a₂ + a₃ = 1**

检验各选项：
- A: 1/2 + 1/2 = 1 ✓
- B: 1/4 + 3/4 = 1 ✓
- C: 2/3 + 1/3 = 1 ✓
- D: 1/3 + 1/3 + 1/3 = 1 ✓

**计算方差：**

由于X₁, X₂, X₃独立同分布，D(Xᵢ) = σ²，所以：
```
D(a₁X₁ + a₂X₂ + a₃X₃) = a₁²σ² + a₂²σ² + a₃²σ²
                        = σ²(a₁² + a₂² + a₃²)
```

**选项A：**
```
D = σ²[(1/2)² + (1/2)²]
  = σ²(1/4 + 1/4)
  = σ²/2
```

**选项B：**
```
D = σ²[(1/4)² + (3/4)²]
  = σ²(1/16 + 9/16)
  = σ²(10/16)
  = 5σ²/8
```

**选项C：**
```
D = σ²[(2/3)² + (1/3)²]
  = σ²(4/9 + 1/9)
  = σ²(5/9)
```

**选项D：**
```
D = σ²[(1/3)² + (1/3)² + (1/3)²]
  = σ²(1/9 + 1/9 + 1/9)
  = σ²(3/9)
  = σ²/3
```

**比较大小：**
```
A: σ²/2 = 0.500σ²
B: 5σ²/8 = 0.625σ²
C: 5σ²/9 = 0.556σ²
D: σ²/3 = 0.333σ²  ← 最小！
```

**答案：D**

**🎓 重要结论：**
> 在所有形如 Σaᵢxᵢ 的无偏估计中（满足Σaᵢ=1），**样本均值**（权重相等）的方差最小！
>
> 这就是为什么我们通常用 X̄ = (1/n)ΣXᵢ 作为μ的估计量！

**🔑 方差最小化原理：**
在约束 a₁ + a₂ + ... + aₙ = 1 下，使 a₁² + a₂² + ... + aₙ² 最小的解是：
```
a₁ = a₂ = ... = aₙ = 1/n
```
这可以用**拉格朗日乘数法**或**柯西不等式**证明！

### 计算题1：全概率公式（10分）

**题目：** 一台机床工作状态良好时，产品的合格率是99%，机床发生故障时的产品合格率是50%。假设每次新开机器时机床处于良好状态的概率是95%，问新开机器后生产的第一件产品是合格品的概率是多少？

**📖 考点分析：**
- **核心考点：** 全概率公式、条件概率
- **难度：** ⭐⭐☆☆☆

**💡 详细解析：**

哼，这是全概率公式的经典应用场景！本小姐来教你如何优雅地解决这类问题～(￣ω￣)

**问题分析：**

这是一个**因果关系**问题：
- **因（原因）：** 机床状态（良好/故障）
- **果（结果）：** 产品质量（合格/不合格）
- **求：** 不考虑原因，直接求结果的概率

**设事件：**
- A：机床处于良好状态
- B：产品是合格品

**已知条件：**
- P(A) = 0.95（机床良好的概率）
- P(Ā) = 0.05（机床故障的概率）
- P(B|A) = 0.99（机床良好时合格率）
- P(B|Ā) = 0.50（机床故障时合格率）

**求：** P(B) = ?

**全概率公式：**
```
P(B) = P(A)·P(B|A) + P(Ā)·P(B|Ā)
```

**计算过程：**
```
P(B) = 0.95 × 0.99 + 0.05 × 0.50
     = 0.9405 + 0.0250
     = 0.9655
     = 96.55%
```

**答案：** 新开机器后生产的第一件产品是合格品的概率为 **0.9655** 或 **96.55%**。

**🎓 全概率公式的使用场景：**

全概率公式适用于：
1. 事件B的发生有多个可能的"原因"A₁, A₂, ..., Aₙ
2. 这些原因构成样本空间的一个**完备事件组**（互不相容且和为全集）
3. 已知P(Aᵢ)和P(B|Aᵢ)，求P(B)

**公式形式：**
```
P(B) = Σ P(Aᵢ)·P(B|Aᵢ)
```

**🔑 解题步骤：**
1. 识别"原因"事件（完备事件组）
2. 列出已知的先验概率P(Aᵢ)和条件概率P(B|Aᵢ)
3. 应用全概率公式求和

**拓展：如果要求"已知产品合格，机床良好的概率"，就要用贝叶斯公式了！**

**方法一：分布函数法（最保险）**

**Step 1：** 求Y的分布函数Fᵧ(y)

```
Fᵧ(y) = P{Y ≤ y}

Fₓ(x) = ∫₀ˣ (2t² + 1/3)dt
      = [2t³/3 + t/3]₀ˣ

```

所以：
```
       { 0,                    y ≤ 5
Fᵧ(y) = { 2((y-5)/4)³/3 + ((y-5)/4)/3,  5 < y < 9
       { 1,                    y ≥ 9
```

**Step 3：** 求fᵧ(y) = F'ᵧ(y)

当5 < y < 9时：
```
fᵧ(y) = d/dy[2((y-5)/4)³/3 + ((y-5)/4)/3]

**方法二：公式法（更快捷）**

其中h = g⁻¹是反函数。

**Step 1：** 求反函数
```
y = 4x + 5
x = (y-5)/4 = h(y)
```

**Step 2：** 求导数
```
h'(y) = 1/4
```

**Step 3：** 代入公式

当x ∈ (0, 1)时，y = 4x + 5 ∈ (5, 9)

```
fᵧ(y) = fₓ((y-5)/4)·|1/4|

      = (y-5)²/32 + 1/12,  5 < y < 9
```

**最终答案：**
```
       { (y-5)²/32 + 1/12,  5 < y < 9
fᵧ(y) = {
       { 0,                  其他
```

**验证（归一性检验）：**

1. g(x)单调（单调增或单调减）
2. g(x)可导

       { 0,                  其他
```

### 计算题3：联合密度、边缘密度、数学期望（10分）

**题目：** 设(X,Y)的联合密度函数为 f(x,y) = { x² + xy/3, 0 ≤ x ≤ 1, 0 ≤ y ≤ 2; 0, 其他 }
求：(1) 关于Y的边缘概率密度函数；(2) E(Y²)

**📖 考点分析：**
- **核心考点：** 联合密度、边缘密度、数学期望的计算
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

哼，二维随机变量是概率论的重点，本小姐来给你详细讲解～(￣ω￣)

**知识回顾：**

**边缘密度函数：**
```
fₓ(x) = ∫₋∞^(+∞) f(x,y)dy
fᵧ(y) = ∫₋∞^(+∞) f(x,y)dx
```

**数学期望：**
```
E(g(X,Y)) = ∫∫ g(x,y)f(x,y)dxdy
```

---

**第(1)问：求fᵧ(y)**

**Step 1：** 确定y的取值范围

从联合密度的定义域看，y ∈ [0, 2]

**Step 2：** 对x积分

当0 ≤ y ≤ 2时：
```
fᵧ(y) = ∫₀¹ (x² + xy/3)dx
      = [x³/3 + x²y/6]₀¹
      = 1/3 + y/6
```

**答案：**
```
       { 1/3 + y/6,  0 ≤ y ≤ 2
fᵧ(y) = {
       { 0,          其他
```

**验证：**
```
∫₀² (1/3 + y/6)dy = [y/3 + y²/12]₀²
                  = 2/3 + 4/12
                  = 2/3 + 1/3
                  = 1  ✓
```

---

**第(2)问：求E(Y²)**

**方法一：利用联合密度**

```
E(Y²) = ∫∫ y²·f(x,y)dxdy
      = ∫₀² ∫₀¹ y²(x² + xy/3)dxdy
      = ∫₀² y² [∫₀¹ (x² + xy/3)dx] dy
      = ∫₀² y²(1/3 + y/6)dy
      = ∫₀² (y²/3 + y³/6)dy
      = [y³/9 + y⁴/24]₀²
      = 8/9 + 16/24
      = 8/9 + 2/3
      = 8/9 + 6/9
      = 14/9
```

**方法二：利用边缘密度**

```
E(Y²) = ∫₋∞^(+∞) y²·fᵧ(y)dy
      = ∫₀² y²(1/3 + y/6)dy
      = ∫₀² (y²/3 + y³/6)dy
      = [y³/9 + y⁴/24]₀²
      = 8/9 + 2/3
      = 14/9
```

**答案：** E(Y²) = **14/9**

**🎓 解题技巧：**

**求边缘密度：**
1. 明确另一个变量的积分范围（可能与当前变量有关）
2. 对另一个变量积分，保留当前变量
3. 注意积分限的确定

**求数学期望：**
- 如果只涉及一个变量的函数E(g(Y))，用边缘密度更简单
- 如果涉及两个变量E(g(X,Y))，必须用联合密度

**🔑 常用公式：**
```
E(Y) = ∫ y·fᵧ(y)dy
E(Y²) = ∫ y²·fᵧ(y)dy
D(Y) = E(Y²) - [E(Y)]²
```

### 计算题4：中心极限定理的应用（10分）

**题目：** 一位职工每天乘公交车上班，如果每天用于等车的时间服从数学期望和标准差均为5min的分布，试用中心极限定理估算他在100个工作日中用于上班的等车时间之和在400min到600min之间的概率。（结果用Φ表示）

**📖 考点分析：**
- **核心考点：** 中心极限定理（独立同分布情形）、标准化
- **难度：** ⭐⭐⭐☆☆

**💡 详细解析：**

哼，中心极限定理是概率论中最重要的定理之一！本小姐来告诉你它的强大之处～(￣▽￣)／

**中心极限定理（Lindeberg-Lévy）：**

设X₁, X₂, ..., Xₙ独立同分布，E(Xᵢ) = μ, D(Xᵢ) = σ²，则：
```
Σᵢ₌₁ⁿ Xᵢ 近似服从 N(nμ, nσ²)  （n充分大时）
```

或标准化后：
```
(ΣXᵢ - nμ)/(σ√n) 近似服从 N(0, 1)
```

---

**已知条件：**
- 每天等车时间Xᵢ：E(Xᵢ) = 5, D(Xᵢ) = 5²=25，σ = 5
- 工作日数：n = 100
- 求：P{400 ≤ ΣXᵢ ≤ 600}

**等等！** 让本小姐重新读题...题目说"标准差均为5min"...

**修正已知条件：**
- E(Xᵢ) = μ = 5 min
- σ = 5 min（标准差）
- σ² = 25 min²（方差）
- n = 100

---

**解题步骤：**

**Step 1：** 确定ΣXᵢ的近似分布

由中心极限定理：
```
X = Σᵢ₌₁¹⁰⁰ Xᵢ ~ N(100×5, 100×25) = N(500, 2500)
```

即：
- E(X) = 500
- D(X) = 2500
- σ(X) = √2500 = 50

**Step 2：** 标准化

```
P{400 ≤ X ≤ 600} = P{(400-500)/50 ≤ (X-500)/50 ≤ (600-500)/50}
                  = P{-100/50 ≤ Z ≤ 100/50}
                  = P{-2 ≤ Z ≤ 2}
```
其中Z ~ N(0, 1)

**Step 3：** 用Φ表示

```
P{-2 ≤ Z ≤ 2} = Φ(2) - Φ(-2)
              = Φ(2) - [1 - Φ(2)]
              = 2Φ(2) - 1
```

**答案：** P{400 ≤ X ≤ 600} = **2Φ(2) - 1**

或写成：**Φ(2) - Φ(-2)**

**数值参考：** Φ(2) ≈ 0.9772，所以概率约为 0.9544 或 95.44%

---

**🎓 中心极限定理的意义：**

1. **普适性：** 无论原始分布是什么，和的分布都趋于正态！
2. **实用性：** 只需知道均值和方差，不需要知道具体分布
3. **近似性：** n ≥ 30时近似效果就很好了

**🔑 解题步骤：**
1. 识别独立同分布的随机变量
2. 计算和的均值：E(ΣXᵢ) = nμ
3. 计算和的方差：D(ΣXᵢ) = nσ²
4. 标准化后查标准正态分布表

**⚠️ 注意事项：**
- 标准差σ和方差σ²要区分清楚
- 标准化时分母是√(nσ²) = σ√n
- Φ(x)表示标准正态分布的分布函数

### 计算题5：矩估计（10分）

**题目：** 设总体X的概率密度为 f(x;θ) = { 2θ²/x³, x ≥ θ; 0, 其他 }，其中参数θ(>0)未知，X₁, X₂, ..., Xₙ是来自总体X的简单随机样本，X̄是样本均值。求参数θ的矩估计量。

**📖 考点分析：**
- **核心考点：** 矩估计法、数学期望的计算
- **难度：** ⭐⭐⭐⭐☆

**💡 详细解析：**

哼，矩估计是参数估计的基本方法！本小姐来教你如何优雅地求解～(￣ω￣)

**矩估计法的原理：**

用**样本矩**代替**总体矩**，建立方程求解参数。

**一阶矩（均值）：** E(X) = μ₁
**二阶矩：** E(X²) = μ₂

**矩估计步骤：**
1. 计算总体的k阶矩（用参数表示）
2. 令样本k阶矩 = 总体k阶矩
3. 解方程组得到参数估计量

---

**解题过程：**

**Step 1：** 计算总体的一阶矩E(X)

```
E(X) = ∫θ^(+∞) x · (2θ²/x³)dx
     = ∫θ^(+∞) (2θ²/x²)dx
     = 2θ² ∫θ^(+∞) x⁻²dx
     = 2θ² · [-x⁻¹]θ^(+∞)
     = 2θ² · [0 - (-1/θ)]
     = 2θ² · (1/θ)
     = 2θ
```

**Step 2：** 建立矩估计方程

根据矩估计法：
```
E(X) = X̄
2θ = X̄
```

**Step 3：** 解出θ的估计量

```
θ̂ = X̄/2
```

**答案：** θ的矩估计量为 **θ̂ = X̄/2**

---

**🎓 矩估计法的特点：**

**优点：**
- 思想简单直观
- 适用范围广
- 计算相对简单

**缺点：**
- 不一定是最优估计
- 有时估计量不唯一
- 可能不满足参数的取值范围

**🔑 常用的矩：**
- **一阶原点矩：** μ₁ = E(X)
- **二阶原点矩：** μ₂ = E(X²)
- **二阶中心矩：** μ₂' = E[(X-μ)²] = D(X)

**一般策略：**
- 1个参数：用一阶矩
- 2个参数：用一阶矩和二阶矩（或方差）

**🔍 本题的扩展：**

如果要求最大似然估计，会更复杂：
```
L(θ) = ∏ᵢ₌₁ⁿ (2θ²/Xᵢ³) · I(Xᵢ ≥ θ)
```
由于θ出现在定义域中，最大似然估计会是：
```
θ̂_MLE = min{X₁, X₂, ..., Xₙ}  （样本最小值）
```

这与矩估计不同！

---

**假设检验的步骤（五步法）：**

**第一步：建立假设**

- H₀（原假设）：μ = 6
- H₁（备择假设）：μ ≠ 6

（这是**双侧检验**）

**第二步：选择检验统计量**

由于σ已知（σ = 0.5），用**Z检验**：
```
Z = (X̄ - μ₀) / (σ/√n) ~ N(0, 1)
```

**第三步：确定拒绝域**

显著性水平α = 0.05，双侧检验：
```
|Z| > Z_{α/2} = Z_{0.025}
```

由Φ(1.96) = 0.975，得：
```
P{Z > 1.96} = 1 - 0.975 = 0.025
```

所以 Z_{0.025} = 1.96

**拒绝域：** |Z| > 1.96

**第四步：计算检验统计量的值**

**Step 4.1：** 计算样本均值
```
X̄ = (5.9 + 6.2 + 6.3 + 6.5 + 6.6 + 6.3 + 6.4 + 6.6 + 5.9) / 9
  = 56.7 / 9
  = 6.3
```

**Step 4.2：** 计算Z值
```
Z = (X̄ - μ₀) / (σ/√n)
  = (6.3 - 6) / (0.5/√9)
  = 0.3 / (0.5/3)
  = 0.3 / (1/6)
  = 0.3 × 6
  = 1.8
```

**第五步：做出判断**

由于 |Z| = 1.8 < 1.96，**不落在拒绝域内**

**结论：** 在显著性水平α = 0.05下，**没有充分理由拒绝原假设H₀**，可以认为该清漆的平均干燥时间为6个小时。

---

**完整答案：**

```
解：(1) 建立假设
    H₀: μ = 6
    H₁: μ ≠ 6

(2) 选择检验统计量
    由于σ² = 0.5² = 0.25已知，用Z检验
    Z = (X̄ - μ₀)/(σ/√n) ~ N(0,1)

(3) 确定拒绝域
    显著性水平α = 0.05，双侧检验
    拒绝域：|Z| > Z_{α/2} = 1.96

(4) 计算统计量的值
    X̄ = 56.7/9 = 6.3

    Z = (6.3 - 6)/(0.5/√9)
      = 0.3/(0.5/3)
      = 0.3 × 6
      = 1.8

(5) 判断
    |Z| = 1.8 < 1.96，不落在拒绝域内

    结论：在α = 0.05的显著性水平下，接受H₀，
         可以认为该清漆的平均干燥时间为6个小时。
```

**🎓 假设检验的核心思想：**

**小概率原理：** 小概率事件在一次试验中几乎不可能发生。

如果在H₀为真的前提下，观察到的结果属于小概率事件，则拒绝H₀。

**🔑 正态总体均值检验的选择：**

| 条件 | 统计量 | 分布 |
|------|--------|------|
| σ²已知 | Z = (X̄-μ₀)/(σ/√n) | N(0,1) |
| σ²未知 | t = (X̄-μ₀)/(S/√n) | t(n-1) |

**双侧检验 vs 单侧检验：**

- **双侧检验：** H₁: μ ≠ μ₀（拒绝域在两侧）
- **右侧检验：** H₁: μ > μ₀（拒绝域在右侧）
- **左侧检验：** H₁: μ < μ₀（拒绝域在左侧）

**⚠️ 两类错误：**
- **第一类错误（弃真）：** H₀为真却拒绝，概率为α
- **第二类错误（取伪）：** H₀为假却接受，概率为β

显著性水平α控制的是第一类错误的概率！
